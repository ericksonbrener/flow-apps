# Flow Apps

Aplicativos web instaláveis (PWA) do **Ecossistema Flow**, hospedados via GitHub Pages.

## Apps

| App | Descrição | Link |
|-----|-----------|------|
| **FinFlow** | Controle de despesas mensais | [`/finflow/`](https://ericksonbrener.github.io/flow-apps/finflow/) |

## Como instalar no celular

1. Abra o link do app no navegador (no **iPhone use o Safari**).
2. Menu do navegador → **Adicionar à tela inicial**.
3. O app abre em tela cheia e funciona offline. Os dados ficam salvos no próprio aparelho.

## Desenvolvimento local

Cada app é uma pasta estática autossuficiente. Para testar:

```bash
cd finflow
node serve.js   # http://localhost:8080
```
